プライバシーポリシー

本プライバシーポリシーは、アプリ「たべメモ」（以下「本アプリ」）における個人情報・データの取り扱いについて説明します。本アプリはアカウント登録を必要とせず、匿名で利用できます。

1. カロリー推定のためにサーバへ送信する情報

本アプリは、あなたが入力した食事の内容（テキスト。音声入力の場合はテキストに変換した文字列）を、カロリーを推定するために、開発者が管理するサーバー（tabememo-api / Vercel 上で稼働）を経由して Anthropic の Claude API に送信します。

  • 送信する内容は食事に関する入力テキストのみで、氏名・連絡先・アカウント等の個人情報とは紐づきません（匿名）。
  • サーバーは推定結果を返すために処理するだけで、入力した本文をデータベース等に保存しません。
  • 秘密鍵・パスワード・第三者の個人情報など、食事と関係のない機密情報は入力しないでください。

2. 端末内に保存する情報

以下の情報は、原則としてあなたの端末内にのみ保存されます。開発者はアクセスできません。

  • 食事の記録（品目・カロリー・記録日時・食事種別）
  • 1日の目標カロリー
  • からだの情報（身長・体重・年齢・性別）— 目標カロリーの自動計算に使用します

iCloud 同期を有効にした場合、これらのデータは Apple の iCloud（CloudKit）にも保存されます。iCloud 上のデータは Apple の管理下にあり、開発者はアクセスできません。

3. 不正利用の防止（App Attest）

サーバーの不正利用を防ぐため、Apple の App Attest（アプリ／端末が正規のものであることを匿名で証明する仕組み）と、端末単位の利用回数制限を使用します。これらは個人を特定するものではなく、広告識別子（IDFA）でもありません。

4. マイク・音声認識について

音声で食事を入力する機能では、Apple の音声認識を使用して音声をテキストに変換します。この処理のために、音声が Apple に送信される場合があります。音声入力を使わない場合、マイクは使用されません。

5. 広告・アクセス解析について

本アプリは、広告の表示や、行動を追跡するアクセス解析を行いません。

6. 利用する外部サービス

  • Anthropic（Claude API） — カロリーの推定
  • Apple — App Attest（不正防止）、音声認識、iCloud 同期
  • Vercel — サーバーのホスティング・実行基盤

各外部サービスでの情報の取り扱いは、それぞれのプライバシーポリシーに従います。

7. データの削除

本アプリをアンインストールすると、端末内に保存されたデータは削除されます。iCloud に保存されたデータは、iOS の設定（iCloud）から削除できます。サーバーは入力本文を保持しないため、サーバー側に削除すべき個人データは残りません。

8. 第三者へのデータ提供

開発者は、法令に基づく場合を除き、あなたの情報を第三者に販売・提供することはありません。

9. 本ポリシーの変更

本ポリシーは必要に応じて更新することがあります。変更後のポリシーはこのページに掲載します。
